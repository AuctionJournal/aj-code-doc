# Cost to become a bidder

There is **no Auction Journal platform fee** to **sign up** as a bidder or to complete **verified bidder** setup. Registration and verification flows do not charge a separate “membership” or “activation” price in the product.

**User-facing guide:** [`user_side_doc/bidder/cost.md`](../user_side_doc/bidder/cost.md)

---

## What is free

| Action | Charge from Auction Journal |
|--------|----------------------------|
| **Bidder registration** (signup, SMS OTP, profile) | None — see [Registration](./registration.md) |
| **Verified bidder** (ID upload + save card via Stripe SetupIntent) | None — see [Verification](./verification.md) |
| **Creating a bidder account** | None |

---

## What is not a “signup fee”

| Item | Meaning |
|------|---------|
| **Credit card on verification** | Used to prove identity and keep a payment method on file for auctions that require **Verified Bidder** registration. Saving the card is **not** a charge to join Auction Journal; Stripe stores the method for later auction-related use. |
| **Winning bids, buyer premium, taxes** | Set by each **auction** and seller/auctioneer rules — separate from account creation. |
| **Listing/auctioneer fees** | Paid by **auctioneers**, not bidders signing up. |
